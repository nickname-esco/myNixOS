📁 Config Directory

The config/ directory contains configuration files for various tools used in this setup:

nvf.bak
Backup of a previous NVF configuration.
ohmyposh/
Theme and prompt configuration for Oh My Posh:
base.json – base theme configuration
zen.toml – alternative theme variant
wezterm/
Terminal configuration:
.wezterm.lua – WezTerm configuration file
zsh/
Shell configuration:
zshrc – Zsh configuration file
